### 중간 복잡도

6. Insurance
   - 딜러 공개 카드가 A일 때 보험 선택.
   - 상태가 하나 늘지만 스플릿보다는 훨씬 작습니다.
   - Even Money와도 자연스럽게 연결됩니다.
7. Dealer Peek
   - 딜러 공개 카드가 A 또는 10점 카드일 때 블랙잭 여부를 먼저 확인.
   - 현재는 stand 시점에 딜러 블랙잭을 확인하는데, 이 룰을 넣으면 라운드 초반 구조가 더 정통 블랙잭에 가까워집니다.
8. Soft 17 룰
   - 딜러가 A+6, 즉 soft 17에서 멈출지 히트할지 선택.
   - 예:
     - Easy House: 딜러 soft 17 stand
     - Hard House: 딜러 soft 17 hit
   - 난이도 모드로 좋습니다.
9. Double Down 제한/확장
   - 현재는 최초 2장 + 자금 충분이면 가능.
   - 변형:
     - 9, 10, 11에서만 가능
     - 어떤 두 장에서도 가능
     - 더블다운 승리 시 특정 카드면 추가 보너스
10. Late Surrender / Early Surrender

- 지금 Fold는 사실상 surrender입니다.
- 변형:
  - Early: 딜러 블랙잭 확인 전 폴드 가능
  - Late: 딜러 블랙잭이면 폴드 불가
- 현재는 캐주얼하게 유지해도 됩니다.

### 높은 복잡도

11. Split

- 가장 큰 구조 변경입니다.
- 플레이어 손패가 여러 개가 되고, 각 손패마다 베팅/결과가 필요합니다.
- 재미는 크지만 구현 전 hand 구조 리팩터링을 추천합니다.

12. Re-split

- 스플릿 후 또 같은 카드가 나오면 다시 스플릿.
- 구현 복잡도 급상승.

13. Split Aces 제한

- A를 스플릿하면 각 손패에 카드 1장만 받고 종료.
- 정통 룰 느낌은 나지만 예외가 늘어납니다.

14. Side Bets

- 메인 블랙잭과 별개로 베팅하는 보너스 룰.
- 예:
  - Perfect Pair
  - 21+3
  - Lucky Ladies
- “1000원으로 1억” 컨셉에는 잘 맞지만, 베팅 UI가 복잡해집니다.

15. Progressive Jackpot

- 특정 희귀 조합이 나오면 초대형 보너스.
- 예:
  - 스페이드 A + 스페이드 J 블랙잭: 100배
  - 원아이드 잭 2장 + 21: 50배
- 하우스를 터는 목표에 잘 맞습니다.
