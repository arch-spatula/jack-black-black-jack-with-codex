# 잭블랙의 블랙잭 게임 디자인 문서

## 게임 개요

- 베팅하는 캐주얼 카드 게임
- 장르: 카드, 트럼프 카드, 갬블링
- 플랫폼: 로컬
  - 누구나 이 프로젝트의 실행파일을 다운받으면 실행할 수 있음
